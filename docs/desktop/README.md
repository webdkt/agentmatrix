# AgentMatrix Desktop 文档

欢迎来到 AgentMatrix Desktop 应用文档。本文档集帮助开发者理解和使用 Desktop 应用的设计系统、组件库和交互流程。

## 📚 快速导航

### 核心文档

| 文档 | 描述 | 目标受众 |
|------|------|---------|
| **[UI设计系统](./ui-design-system.md)** | 完整的设计规范，包括颜色、字体、间距、组件样式等 | 设计师和前端开发者 |
| **[组件参考](./component-reference.md)** | 组件目录、Store状态、API接口、概念映射表 | 前端开发者 |
| **[交互设计](./ui-interaction-flows.md)** | 特殊功能流程、状态管理、WebSocket通信机制 | 前端开发者和产品经理 |

### 使用指南

#### 1️⃣ 新手开发者
建议按以下顺序阅读：
1. [UI设计系统](./ui-design-system.md) - 了解设计规范和基本概念
2. [组件参考](./component-reference.md) - 熟悉组件结构和状态管理
3. [交互设计](./ui-interaction-flows.md) - 理解特殊的交互流程

#### 2️⃣ 功能开发
如果你正在开发新功能：
1. [组件参考](./component-reference.md) - 查找可复用的组件和Store
2. [交互设计](./ui-interaction-flows.md) - 了解如何处理特殊流程（如Agent问答）
3. [UI设计系统](./ui-design-system.md) - 确保UI符合设计规范

#### 3️⃣ UI调整
如果你正在进行UI优化：
1. [UI设计系统](./ui-design-system.md) - 查看设计token和组件规范
2. [组件参考](./component-reference.md) - 了解组件的Props和Events
