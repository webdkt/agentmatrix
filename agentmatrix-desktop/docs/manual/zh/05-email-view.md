# 邮件视图

邮件视图是 AgentMatrix Desktop 的主要界面 —— 一个三栏布局，用于管理所有智能体通信。

## 布局

邮件视图从左到右包含三个面板：

1. **视图选择器** —— 左侧导航栏，包含各种图标（邮件、Matrix、设置等）
2. **会话列表** —— 显示所有对话会话的侧边栏
3. **邮件列表** —— 显示所选会话中邮件的主区域

## 阅读邮件

从会话列表中选择一个会话以查看其邮件。每封邮件卡片显示：

- **发件人名称** —— 发送邮件的智能体或用户
- **时间戳** —— 邮件发送时间
- **主题行** —— 邮件话题
- **正文** —— 消息内容，支持 Markdown 渲染
- **附件** —— 邮件附带的文件

邮件正文支持完整的 Markdown 渲染，包括：

- 标题、加粗、斜体、删除线
- 带语法高亮的代码块
- 表格、列表、引用块
- 链接和图片

## 撰写新邮件

点击 **新建邮件** 按钮打开撰写对话框。您需要指定：

- **收件人** —— 从下拉菜单选择目标智能体
- **主题** —— 任务的描述性标题
- **正文** —— 您的任务描述或问题
- **附件** —— 可选择附加文件

点击 **发送** 将邮件分发给智能体。

## 回复邮件

每封邮件都有 **回复** 按钮，点击后打开回复输入区域：

- 回复自动关联到当前会话
- 您可以直接在邮件下方输入回复
- 回复中可以包含文件附件
- 按发送按钮或使用键盘快捷键分发

## 删除邮件

您可以从列表中删除单个邮件：

- 点击邮件卡片上的删除按钮
- 在对话框中确认删除
- 已删除的邮件会被永久移除

## 搜索

会话列表包含一个搜索栏，可按名称或参与者过滤会话。输入关键词即可快速找到特定对话。

## 实时更新

邮件视图通过 WebSocket 实时更新：

- 新邮件自动出现，无需刷新
- 智能体状态变化立即反映
- 新邮件到达时会收到桌面通知（如已启用）

## 键盘快捷键

| 操作 | 快捷键 |
|------|--------|
| 发送回复 | Cmd/Ctrl + Enter |
| 新建邮件 | Cmd/Ctrl + N |
| 搜索会话 | Cmd/Ctrl + K |
