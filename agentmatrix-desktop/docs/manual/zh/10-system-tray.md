# 系统托盘与通知

AgentMatrix Desktop 通过系统托盘图标提供快速访问和后台运行功能。

## 系统托盘图标

AgentMatrix Desktop 运行时，托盘图标会出现在系统的菜单栏（macOS）或任务栏通知区域（Windows/Linux）。

### 托盘菜单

右键点击（macOS 上直接点击）托盘图标可访问：

| 操作 | 说明 |
|------|------|
| **打开 AgentMatrix** | 显示并聚焦应用窗口 |
| **退出** | 完全关闭应用 |

直接点击托盘图标会显示/隐藏应用窗口。

## 窗口行为

- **关闭按钮** —— 最小化到托盘（不会退出应用）
- **点击托盘图标** —— 切换窗口可见性
- **从托盘退出** —— 完全退出应用

## 桌面通知

AgentMatrix Desktop 可以为重要事件显示原生桌面通知：

- **收到新邮件** —— 当智能体发送回复时
- **智能体状态变化** —— 当智能体启动、停止或遇到错误时
- **后端状态** —— 当 Python 后端启动或停止时

### 启用 / 禁用通知

可以通过应用设置控制通知：

- 在 `~/.agentmatrix/settings.json` 中将 `enable_notifications` 设置为 `true` 或 `false`
- 默认为 `true`（通知已启用）

首次启动时，系统可能会提示您授予 AgentMatrix 通知权限。

### 通知降级

如果原生通知不可用（如权限被拒），AgentMatrix 会降级为应用内浏览器风格的通知。
