# Matrix 视图

Matrix 视图是智能体监控仪表板 —— 实时查看 Matrix World 中所有智能体的状态。

## 进入 Matrix 视图

点击视图选择器（左侧栏）中的 **Matrix** 图标切换到 Matrix 视图。

## 智能体列表

左侧面板显示所有已配置的智能体及其当前状态：

| 状态 | 说明 |
|------|------|
| **运行中** | 智能体正在积极处理任务 |
| **空闲** | 智能体在线但未在处理 |
| **已停止** | 智能体未运行 |
| **启动中** | 智能体正在初始化 |
| **停止中** | 智能体正在关闭 |

每个智能体条目显示：

- **智能体名称** —— 配置的名称
- **状态指示器** —— 显示当前状态的彩色圆点
- **最后活动** —— 智能体上次处理任务的时间

使用搜索栏按名称过滤智能体。

## 智能体仪表板

从列表中选择一个智能体以查看其仪表板。仪表板包含多个标签页：

### 档案标签

显示智能体的配置：

- **名称** —— 智能体标识符
- **描述** —— 智能体的功能
- **类** —— 智能体类型
- **技能** —— 能力列表
- **后端模型** —— 智能体使用的 LLM/SLM
- **人格** —— 智能体的角色定义（预览）

### 资源标签

显示智能体可以访问的资源：

- **浏览器** —— 网页浏览能力
- **计算机** —— 系统命令访问
- **主文件夹** —— 智能体的工作目录
- **会话文件夹** —— 会话特定的数据目录
- **技能文件夹** —— 可用的技能模块

### 日志标签

实时日志查看器，显示：

- 智能体思考过程
- 动作执行
- 工具调用
- 错误消息
- 系统事件

日志在智能体工作时实时更新。

### 记忆标签

显示智能体的记忆状态 —— 它在任务中积累和保留的信息。

## 智能体操作

在智能体仪表板中，您可以对智能体执行以下操作：

| 操作 | 说明 |
|------|------|
| **查看提示词** | 查看智能体的完整系统提示词 |
| **停止** | 停止智能体（终止当前任务） |
| **暂停** | 暂停智能体（挂起处理） |
| **恢复** | 恢复已暂停的智能体 |
| **重载** | 重新加载智能体的配置 |
| **克隆** | 创建智能体的副本，使用新名称 |
| **发送邮件** | 撰写新邮件给此智能体 |

## 会话历史

仪表板还显示智能体的会话历史 —— 智能体参与的所有会话列表，包含时间戳和状态。
