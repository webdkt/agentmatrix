# 首次运行向导

首次启动 AgentMatrix Desktop 时，引导式设置向导会带您完成必要的配置。向导采用 Matrix 雨动画背景，包含六个步骤。

## 第 1 步：欢迎

欢迎屏幕介绍 AgentMatrix Desktop。点击 **开始** 进入设置流程。

## 第 2 步：您的名字

输入您的显示名称。此名称将用于整个系统中标识您的身份 —— 在会话、邮件和智能体交互中。

- 使用有意义的名称（例如您的真实姓名或一致的别名）
- 后续可以通过编辑系统配置文件来修改

## 第 3 步：工作空间目录

选择一个计算机上的目录作为 **Matrix World** —— 存储所有智能体数据、配置和文件的工作空间。

- 点击目录选择器浏览并选择文件夹
- 默认建议为 `~/MatrixWorld`
- 如果目录不存在，会自动创建
- 内部会创建 `.matrix/configs/` 子目录，包含所有配置模板

**重要提示**：请选择具有足够磁盘空间的目录。智能体文件、日志和浏览器数据会随时间累积。

## 第 4 步：大脑（大语言模型）

配置驱动智能体推理的主要 AI 模型。这是"大脑" —— 用于复杂思考和决策的强大模型。

从可用的提供商中选择：

| 提供商 | 推荐模型 |
|--------|----------|
| **Anthropic** | Claude Sonnet 4, Claude Haiku 4 |
| **OpenAI** | GPT-4o, GPT-4o Mini, o1, o3 |
| **DeepSeek** | DeepSeek Chat, DeepSeek Coder |
| **Google** | Gemini 2.0 Flash, Gemini 1.5 Pro |
| **小米** | MiMo v2 Pro, MiMo v2 Flash |
| **自定义** | 任何 OpenAI 兼容的 API 端点 |

每个提供商需要提供：

- **API Key** —— 您的提供商 API 密钥（本地保存，绝不共享）
- **模型名称** —— 要使用的具体模型
- **API URL** —— 已知提供商自动填充，自定义端点需手动输入

## 第 5 步：小脑（小语言模型）

配置一个更小、更快的模型，用于轻量级任务，如邮件解析、简单分类和快速响应。这释放主模型用于复杂推理。

- 相同的提供商选择
- 选择更小/更快的模型（如 GPT-4o Mini, Claude Haiku, Gemini Flash）
- 可以使用与大脑相同或不同的提供商

## 第 6 步：检查并启动

在启动前检查所有设置：

- **用户名** —— 您选择的显示名称
- **数据目录** —— Matrix World 工作空间路径
- **大模型** —— 大脑的提供商和模型
- **小模型** —— 小脑的提供商和模型

您还可以选择配置 **邮件代理** 设置：

- **启用邮件代理** —— 开关，用于将 AgentMatrix 连接到真实邮箱
- **Matrix 邮箱** —— Matrix 系统的邮箱地址
- **用户邮箱** —— 您的个人邮箱地址
- **IMAP 设置** —— 收件服务器（主机、端口）
- **SMTP 设置** —— 发件服务器（主机、端口）

点击 **启动** 初始化您的 Matrix World。向导将：

1. 创建工作空间目录
2. 复制配置模板
3. 保存 LLM 配置
4. 启动 Python 后端
5. 过渡到主应用界面

## 重新运行向导

向导会在以下情况自动出现：

- 设置文件（`~/.agentmatrix/settings.json`）不存在
- 配置的工作空间目录不存在
- LLM 配置文件缺失

在开发期间强制显示向导：

```bash
./start-dev.sh --force-wizard
```
