# 简介

## 什么是 AgentMatrix Desktop？

AgentMatrix Desktop 是一个**分布式认知调度系统** —— 一个桌面应用程序，让您通过直观的类邮件界面来管理、监控和与自主 AI 智能体交互。

可以把它想象成一个 **AI 工作调度中心**。每个 AI 智能体就像一个拥有自己收件箱的团队成员。您通过邮件发送任务，智能体自主执行，然后通过邮件向您汇报结果。

## 工作原理

1. **您撰写一封邮件**，描述一个任务
2. **后端调度**将任务分发给合适的 AI 智能体
3. **智能体思考与行动** —— 使用工具、浏览网页、写文件
4. **智能体回复**结果邮件
5. **您在邮件视图中**查看结果

## 核心概念

| 概念 | 说明 |
|------|------|
| **智能体 (Agent)** | 一个自主 AI 实体，具有定义好的人格、技能和工具访问权限 |
| **会话 (Session)** | 您与一个或多个智能体之间的对话线程 |
| **邮件 (Email)** | 在会话中发送的消息（任务或回复） |
| **Matrix World** | 存储所有智能体数据、配置和文件的工作空间目录 |
| **LLM / SLM** | 驱动智能体推理的大语言模型和小语言模型 |
| **技能 (Skill)** | 智能体可以使用的能力（网页搜索、文件管理、邮件等） |
| **人格 (Persona)** | 智能体的性格和角色定义 |

## 主要功能

- **首次运行向导** —— 引导式初始设置（用户名、工作空间、AI 模型）
- **邮件界面** —— 发送、接收、回复和管理智能体通信
- **会话管理** —— 通过搜索和分页组织对话
- **Matrix 视图** —— 所有智能体的实时监控仪表板
- **设置面板** —— 无需编辑文件即可配置 AI 模型和邮件代理
- **实时通知** —— WebSocket 驱动的即时更新
- **多语言支持** —— 中文和英文界面

## 支持平台

AgentMatrix Desktop 基于 Tauri 2.0 构建，支持：

- **macOS**（Apple Silicon 和 Intel）
- **Windows**（x64）
- **Linux**（x86_64）
