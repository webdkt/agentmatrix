# Promotion

> Applicable scenarios: Brand promotion, marketing campaigns, product launches, portfolio showcases, creative pitches, personal branding, event promotions, etc.
> Style anchor: High-end brand lookbooks + Apple-style product presentations + creative agency pitch decks

## Design Philosophy
- **Visual impact first**: The first impression is everything; bold, memorable visuals that capture attention immediately
- **Emotional resonance**: Design should evoke desire, excitement, trust, or aspiration — aligning with the brand's emotional positioning
- **Less is more (strategic minimalism)**: Restraint in text density; let visuals, whitespace, and typography carry the message
- **Brand consistency**: Every element must reinforce the brand identity — colors, fonts, imagery style, and tone must be cohesive
- **Information density: Low to Moderate (40-60%)**: Promotion is about persuasion, not exhaustive information; prioritize key messages and visual storytelling

## Text-to-Visual Ratio: Visual-heavy, Text as Supporting Narrative
- Visuals (photography, video stills, product renders, illustrations) are the primary communication vehicle
- Recommended ratio: approximately 30% text + 60% visuals + 10% dramatic whitespace
- Text should be concise, punchy, and serve as captions or headlines to visuals
- Big, bold typography as a visual element itself — large headlines can be the dominant visual

## Color Guidance
- **Bold, confident palette**: Promotion scenarios can afford more expressive color choices than academic or business contexts
- **Brand-aligned primary colors**: The primary color should be directly derived from the brand's visual identity; if no brand exists, choose a color that evokes the desired emotion
  - Luxury: Black, gold, deep navy, burgundy, cream
  - Tech/Future: Deep blue, electric blue, neon accents, black, white
  - Nature/Wellness: Sage green, earth tones, soft blues, warm beiges
  - Youth/Energy: Bright coral, vibrant yellow, electric teal, hot pink
  - Professional trust: Navy, teal, forest green, charcoal
- **High contrast for drama**: Use strong light-dark contrast, complementary color accents for call-to-action elements
- **Gradient and texture allowed**: Subtle gradients, metallic textures, or paper textures may be used for atmosphere (sparingly, not on text backgrounds)
- **Accent colors for CTAs**: Reserve the brightest/most saturated color for call-to-action buttons, key metrics, and emphasis
- **Dark mode friendly**: Promotion decks often benefit from dark backgrounds for cinematic impact; ensure sufficient contrast for all text

## Font Guidance
- **Headlines and display text**: Bold, impactful sans-serif (e.g., Montserrat Black, Bebas Neue, Poppins Bold, Impact) or elegant serif for luxury (e.g., Playfair Display, Libre Baskerville)
- **Body text**: Clean, readable sans-serif (e.g., Open Sans, Lato, MiSans); avoid decorative fonts for paragraphs
- **Chinese**: MiSans or PingFang SC for modern brands; Songti or Source Han Serif for luxury/traditional brands
- Font size hierarchy should be dramatic and attention-grabbing:
  - Cover/hero headline: 56-96px (or larger — oversized typography as art)
  - Section titles: 36-48px
  - Page titles: 28-36px
  - Body text: 18-22px (minimum 18px; promotion text is often sparse, so 20-22px is preferred)
  - Captions, fine print: 14-16px
  - Minimum font size: 12px (only for legal/copyright text)

## Content Page Structure
- **Promotion page structures**:
  - Full-bleed hero image + minimal centered text (brand statement, tagline)
  - Product showcase: Large product image + key feature bullets
  - Data/impact slide: Big numbers (metrics) + brief explanation
  - Testimonial/review: Quote + attribution + brand logo
  - Comparison/competitive advantage: Side-by-side or grid layout
  - Call-to-action (CTA): Clear, isolated CTA with supporting value proposition
- **Cover/chapter pages**: Must be visually arresting; consider full-bleed photography with gradient masks and large typography
- **Whitespace is strategic**: Use generous whitespace to create breathing room and focus attention on key messages

## Narrative Style: Persuasion and Storytelling
- Lead with the audience's needs/desires, not product features
- Structure: Hook (pain point/aspiration) -> Solution (product/service) -> Proof (data/testimonials) -> Call to Action
- Use storytelling: customer journey, before/after, success stories
- Keep text minimal: headlines, subheads, and bullet points; avoid paragraphs
- Every page should answer: "Why should I care?" and "What's in it for me?"
- End with a strong, clear Call to Action (CTA)

## Content Expression Techniques
- **Hero images and full-bleed photography**: Use high-quality, magazine/editorial-level photography; apply gradient masks for text overlay legibility
- **Big number highlights**: Showcase key metrics (users, revenue, growth, satisfaction rate) with oversized typography
- **Product/feature showcases**: Clean product shots on neutral or branded backgrounds; use subtle shadows for depth (if aligned with brand)
- **Quote/testimonial cards**: Large quotation marks, italic or serif font for the quote, clean attribution
- **Timeline/history**: Horizontal or vertical timelines with milestone markers and concise descriptions
- **Iconography**: Consistent, brand-aligned icon set (solid or outline); use for feature lists and navigation
- **Video/stills**: Reference video content with high-quality still frames; add play button overlay for video content

## Image Usage Rules
**Allowed (brand/promotional images)** — High-quality, purposeful imagery:

| Image Type | Examples | Applicable Pages |
|---------|------|---------|
| Brand photography | Product shots, lifestyle photography, model photography | Cover, product pages, portfolio |
| Hero/atmospheric images | Full-bleed landscapes, abstract textures, brand color fields | Cover, chapter transitions, closing |
| Logo lockups and brand assets | Partner logos, certification badges, media mentions | Trust/social proof pages |
| Portfolio/work samples | Case study screenshots, design work, project photos | Portfolio, case study pages |
| Data visualizations | Simplified charts showing growth, reach, impact | Metrics, impact pages |
| User/customer photos | Authentic user testimonials, community photos | Testimonial, community pages |

**Prohibited (low-quality or generic images)**:
- Low-resolution, pixelated, or watermarked images
- Overused stock photos that lack brand alignment
- Clip art or cartoon illustrations for professional/luxury brands
- Unrelated decorative images that do not serve the narrative

## Decoration Guidelines (Allowed and Encouraged)
| Element | Guideline |
|--------|---------|
| Gradients | Allowed for backgrounds, overlays, and text effects; keep subtle and brand-aligned |
| Shadows | Soft, realistic shadows for product images and cards; avoid heavy drop shadows |
| Rounded corners | Modern and friendly; 8-24px depending on brand personality |
| Overlays and masks | Gradient overlays (dark or brand-colored) on images to ensure text legibility |
| Decorative shapes | Abstract geometric shapes, brand pattern elements, subtle textures — used as background accents |
| Typography as decoration | Oversized, partially cropped, or low-opacity text as background texture |

## Decoration Prohibitions
| Prohibited | Alternative |
|--------|---------|
| Cluttered, busy layouts | Single focal point per page |
| Inconsistent image styles | Curate images to a unified color grade, crop style, and mood |
| Excessive text blocks | Chunk text into headlines + bullets; use visuals to replace text where possible |
| Generic clip art or emoji | Custom illustrations or brand-aligned icons |
| Mismatched color schemes | Stick to the defined brand palette |
| Poor contrast (light text on light image) | Always use image masks or text shadows for legibility |

## Additional Promotion-Specific Guidelines
- **Brand consistency audit**: Before generating, confirm the brand's existing visual assets (logo, colors, fonts, photography style) and strictly adhere to them
- **Mobile-first preview**: Many promotion decks are viewed on screens; ensure key text is readable at smaller sizes
- **Animation hints**: If the deck will be presented live, consider layout structures that support reveal animations (staggered bullets, image wipes)
- **CTA design**: Every promotional deck should have at least one strong CTA page; make it visually distinct (different background, centered layout, bold button-style shape)
- **Social proof**: Dedicate pages to logos of clients, media mentions, awards, or testimonials; grid layouts work well here
